## Debug me in local

1. npm install spm -g
2. git clone https://github.com/sorrycc/puzzle
3. cd puzzle
4. spm install && spm doc watch
5. open http://127.0.0.1:8000

More detail in http://aralejs.org/docs/develop-components.html
